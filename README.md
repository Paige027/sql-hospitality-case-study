# SQL Hospitality Case Study

This repository contains a sample hospitality bookings dataset and example SQL queries to extract business insights.

Files:
- `dataset.csv` — sample bookings data
- `analysis.sql` — example queries (aggregations, window functions, joins)
- `revenue_by_roomtype.png` — sample chart of revenue by room type

Usage:
Load `dataset.csv` into your SQL environment (Postgres, BigQuery, SQLite) and run `analysis.sql`.
